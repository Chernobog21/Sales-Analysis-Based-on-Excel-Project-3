# Sales Analysis Based on Excel (Project 3)

This is my first data analysis project where the dataset is based on an Excel spreadsheet and not a Ð¡SV.
The following actions were performed on the data:
- Data clearing
- Checking data completeness and presence of duplicates
- Correlation analysis
- Sales analysis by month
- Sales analysis by category
- Sales analysis by day of the week
- Sales analysis by country
- Visualization of all metrics

The following tools were used in working on this project:
- Python (Pandas, Matplotlib, Seaborn)
- Excel
- Jupiter notebook
- Google slides

An artificially generated dataset was used, but the data was created as close as possible to the real business.

The main goal of this project was to try working with another, more popular data type in the form of Excel tables, as well as to create a more professional report in PDF format.
Despite the artificially generated dataset, the analysis was complete and close to real business conditions.
